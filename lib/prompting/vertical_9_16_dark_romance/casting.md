# Character Casting: Vertical Dark Romance

## Task
Analyze text for KEY characters, locations, rooms, objects, and vehicles and generate photorealistic reference descriptions.
These references will be used for vertical dark romance shots and AI video animation in 9:16 portrait format.
Think as a **Casting Director and Production Designer** for prestige vertical romance — faces carry 80% of the drama.

## Character Description Format
For each NEW character:
- **Name**: Full name or identifier
- **Visual Description**: Photorealistic actor description with specific physical features
  - Face: Facial structure, jaw, eyes (color, shape, expression default), brow weight, lips
  - Hair: Color, texture, style — specific enough to be consistent across scenes
  - Build: Height, physique, posture — how they occupy space
  - Clothing: Default wardrobe register (luxury, professional, casual) with specific fabric textures and colors
  - Age: Specific age or tight range
  - Distinctive features: The one detail that makes them unmistakable — a scar, a ring, a specific way they hold their shoulders

## Dark Romance Character Standard
**Male lead archetype**: Controlled, dangerous stillness. Expensive clothing worn without vanity. Eyes that track rather than dart. A jaw that holds tension invisibly. The sense of someone who never raises their voice because they never need to.
**Female lead archetype**: Composed intelligence. Beauty that reveals itself gradually — the kind that is easy to underestimate and impossible to forget. Posture that says she is not afraid, though she should be.

## Location / Room / Object / Vehicle Format
For each NEW reference:
- Name: Full name
- Visual Description: Photorealistic environment or object description with specific textures, materials, lighting conditions
- Distinctive features: What makes it recognizable and atmospherically correct for dark romance

## Background Rules
- Characters: EMPTY BACKGROUND, neutral deep charcoal or midnight blue
- Locations/rooms: SHOW EMPTY SPACE WITHOUT PEOPLE — but retain all atmospheric lighting (practical lamps on, candles lit, city view through window)
- Objects: BLANK BACKGROUND, single object, luxury quality
- Vehicles, interiors: SHOW EMPTY, WITHOUT PEOPLE — retain interior lighting and texture detail

## Reference Shot Format
- **Shot Type**: Close-up portrait, slight below-eye-level angle (adds authority for male lead; vulnerability for female lead)
- **Expression**: Controlled neutral — with deliberate subtext. Not blank. The face that knows something.
- **Lighting**: Single warm side-key (amber/candlelight quality) + cool blue edge from opposite side (city window). Shadow falls on one cheek — always the same cheek per character for consistency.
- **Background**: Solid deep charcoal or midnight blue. No texture, no distraction.
- **Quality**: 2K resolution, sharp focus, photorealistic skin with visible pores at ECU

## Room Reference Format
For every room, generate a single 2-panel image stacked vertically:
- TOP: View from the doorway into the room — establish full spatial layout, lighting, atmosphere
- BOTTOM: View from inside the room toward the doorway — the escape / the entrance

## Vehicle Reference Format
For every vehicle, generate a single 3-panel image stacked vertically:
- TOP: Exterior — night/luxury setting context
- MIDDLE: Interior from entrance — materials, lighting, spatial intimacy
- BOTTOM: Interior from driver position toward passenger — the angle used in dialogue scenes

## Visual Description Quality
Verbose, precise, cinematographer-grade. Every feature specific enough that an AI model generates the same person/space twice without drift.
Include: exact shade names for colors (not "brown hair" — "dark espresso brown, slight natural wave"), fabric specifics ("charcoal merino wool suit, peak lapel"), spatial details ("180cm², high ceiling, single floor lamp creating a 45° warm cone in the center of the room").
